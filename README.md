## 現在地

このリポジトリは、**ヒデロックテンプレ法©️アプリのデモ版** です。

現時点では、万人向けの完成アプリではありません。

まずは、ヒデロック本人が、人生の素材・違和感・体感・創作ネタ・仕事判断・AIへの依頼文を、**AIと現実に渡せる工程へ変換するための台帳** として作っています。

---

## このアプリの理念

ヒデロックテンプレ法©️アプリは、**人生の素材を、AIと現実で使える工程に変えるための道具** です。

対象そのものを保存するのではなく、**対象を扱うための工程** を保存します。

だから、ジャンルが変わっても使えます。

アプリ開発でも、note記事でも、病院説明でも、クラウドワークス案件でも、ロードー判断でも、ギター練習でも、食べ物ログでも、呼吸体験でも、入院バケーションでも、スパイスと世界史でも、基本の工程は同じです。

## 誰向けか

### 第1対象：ヒデロック本人

このアプリは、まずヒデロック本人用です。

- 変な言葉を拾う
- 違和感を捨てない
- 体感を構造へ変換する
- AIに渡せる形へ圧縮する
- 生活・創作・仕事・AI活用を混ぜたまま放置しない

そのための、外付け工程台帳です。

### 第2対象：似たタイプの人

将来的には、似たタイプの人にも使える可能性があります。

- クリエーター
- アイデアが多い人
- 整理の前で止まりやすい人
- AIに何を頼めばいいかわからない人
- 自分の体感を説明文にするのが苦手な人
- 生活・仕事・創作が混ざりやすい人

ただし、最初から一般向けにしすぎません。

まず本人用として使える芯を作り、その後、人にも渡せる部分だけを分岐させます。

## 基本工程

1. **拾う**  
   変な言葉・違和感・体感・出来事を拾う。

2. **名前をつける**  
   呼べる形にする。

3. **レイヤー分解する**  
   身体、現実、創作、仕事、AI、社会語に分ける。

4. **地上語へ翻訳する**  
   他人やAIに渡せる言葉にする。

5. **方法©️化する**  
   次回も使える工程として保存する。

---

## デモ版でできること

このデモ版では、以下のことができます。

- 素材を入力する
- 5つの基本工程に沿って整理する
- 劇団員検査レイヤーでチェックする
- 最終判定を選ぶ
- 次の一手を書く
- やらないことを書く
- AIに渡す文としてコピーする

外部API、ログイン、データベースは使っていません。  
GitHub Pagesで動く、静的Webアプリです。

## 開発方針

このアプリは、まず **ヒデロック本人用の方法©️台帳** として作ります。

他人向けの一般化は、本人用の芯が動いてから行います。

```text
main
└─ ヒデロック本人用・安定版

feature/theater-tab
└─ 劇団員辞典タブの実験

feature/storage-shelf
└─ 方法©️棚の実験

feature/public-lite
└─ 他人にも使える言葉へ変換する実験
```

---

## ブランチ運用

### main

公開してよい安定版です。  
まずはヒデロック本人用の芯を守ります。

### feature ブランチ

新機能や別方向の実験を行う場所です。

例：

- `feature/theater-tab`
- `feature/storage-shelf`
- `feature/public-lite`

試して良ければ main に合流します。  
違ったら無理に採用しません。

---

## 他人向け版について

他人向け版を作る場合も、いきなり main を変えません。

まずは `feature/public-lite` のようなブランチで、説明・UI・サンプルだけを調整します。

完全に別物になった場合は、別リポジトリに分けることも検討します。

## v0.3.2 メモ

### 空欄OK・素材ワンパン生成パッチ

- 上部に「全部埋めなくてOK」の説明カードを追加
- 「素材ワンパン生成」と「全部入り生成」を追加
- 素材ワンパン生成では未入力項目を省略
- 全部入り生成では未入力を「（未入力）」として表示
- コピーは最後に生成した文を優先
- 未生成でコピーした場合は素材ワンパン生成として処理

## v0.3.3 メモ

### 消えない台帳パッチ

- 入力途中の台帳内容を、この端末のブラウザ内に自動保存
- ページを開き直した時に自動復元
- 「台帳を空にする」ボタンを追加
- 台帳を空にすると、入力欄・出力欄・保存された下書きを削除
- 保存キー：`hiderockTemplateDraftV1`

これは本格的な「方法©️棚」ではありません。  
名前をつけて複数保存する機能は v0.5 以降の候補です。

## v0.4A メモ

### 素材投入型マニュファクチュア工場©️

v0.4Aでは、画面の主役を「手動記入式台帳」から「素材投入型マニュファクチュア工場©️」へ移しました。

- ユーザーは素材投入口に生素材を入れる
- 加工先を選ぶ
- 工場ラインを選ぶ

- 旧5工程カードと旧劇団員入力欄は削除せず、非表示で保持
- コピー、自動保存、自動復元、台帳を空にする機能は継続
